Current Research
  • Lexico-Syntactic Triggers of Language Model Hallucinations
    This study systematically investigates the lexico-syntactic patterns that trigger hallucinations in large language models, creating a taxonomy of high-risk prompt structures and developing validated prompting guidelines to mitigate these effects. It establishes a replicable experimental framework for prompt optimization, contributing concrete tools for responsible AI integration and safety.
